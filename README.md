# 🤖 Doraemon AI Chatbot

An AI conversational chatbot inspired by **Doraemon**, built using **HuggingFace Transformers**, **BlenderBot**, and **Gradio UI**.
The chatbot maintains conversation context and responds in a friendly Doraemon-like personality.

##  Live Demo

[(Insert your HuggingFace Spaces link here)](https://huggingface.co/spaces/Purnapragya/doraemon_chatbot)

## Features

* Conversational AI chatbot
* Doraemon personality simulation
* Context-aware conversation memory
* Interactive web UI using Gradio
* HuggingFace deployment
* Lightweight transformer model

## Tech Stack

* Python
* HuggingFace Transformers
* BlenderBot-400M
* Gradio
* PyTorch

## Model Used

**facebook/blenderbot-400M-distill**

This model is a distilled version of Facebook's BlenderBot designed for conversational AI tasks.

## Project Structure

app.py → Main chatbot application
requirements.txt → Dependencies
README.md → Documentation

## Installation (Local Run)

Clone repository:

git clone https://github.com/Purnapragya12/doraemon_chatbot

cd doraemon-chatbot

Install dependencies:

pip install -r requirements.txt

Run application:

python app.py

## Deployment

This project is deployed using **HuggingFace Spaces with Gradio SDK**.
